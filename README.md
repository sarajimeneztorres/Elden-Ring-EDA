# 🎮 Análisis de Retención en Steam (Elden Ring)

Este proyecto es un Análisis Exploratorio de Datos (EDA) sobre las reseñas de Steam para descubrir patrones de retención de jugadores.

**Herramientas utilizadas:** Python, Pandas, Requests, Seaborn.

**Hallazgo principal:** El fenómeno del "Veterano Frustrado". Los jugadores que NO recomiendan el juego tienen una retención mediana mucho mayor (472h) que los que SÍ lo recomiendan (171h).
